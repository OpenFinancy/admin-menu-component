### Added
- Initial public release of the admin menu component.
- Composite menu contracts (`MenuComponent`, `AbstractMenuComponent`) plus `MenuGroup` and `MenuLink` implementations.
- PHPUnit test suite covering menu grouping and leaf behaviour.
- CI workflow, static analysis, and coding standards tooling.
